- 4-设备定义
    - 智能镭射光逗猫玩具
        - 主要电子元器件
            - 主控
                - 内置蓝牙芯片，负责开机后的业务逻辑控制、数据处理和功能实现
            - 毫米波雷达
                - HLK-LD2450，负责移动目标位置、速度、运动方向的实时检测
                - 可检测范围：6米，方位角±60°，俯仰角±35°
            - 步进电机
                - x 2，可控制镭射光发射器在水平±55°、垂直±30°范围内偏转角度
            - 镭射光发射器/激光灯
                - 光点直径20毫米，颜色为橙红色
        - 安装要求
            - 可以壁挂在墙壁或家具侧壁上，也可以蹲坐在家具/家装顶部平面上
            - 建议距离地面1.5~2.5米
            - 雷达芯片需要前倾40°安装
            - 镭射光柱与地面夹角的默认值为50°
        - 功能定义
            - 配对
                - 设备满足条件1时响应配对请求
                    - 条件1：尚未配对，或由于之前配对过的手机关机/关闭蓝牙/不在通信范围无法连接
                    - 说明：蓝牙未连接时如果之前设置过逗宠区域，则依然能够正常执行自动逗宠，否则不会执行自动逗宠
                - 配对过程
                    - 设备广播含有预设UUID和设备名称
                    - APP扫描到含有预设UUID的设备后，在APP内展示给用户
                    - 用户点击APP上的【配对】按钮，APP向目标设备发起GATT通信建立并得到设备响应
            - 设备安放高度设置
                - 设置目标
                    - 准确计算镭射光发射器的偏转角度和镭射光点位置的关系，使得镭射光点模仿猎物的行为更逼真
                - 进入设置模式
                    - 触发条件：a或b被满足
                        - a
                            - APP通过蓝牙连接通知设备进入，并得到设备确认
                        - b
                            - 安放高度未设置
                    - 光点位置
                        - 镭射光点位于0°点/中心点
                    - 发射状态
                        - 镭射光处于持续发射状态
                    - 指示灯
                        - 设备指示灯进入绿色闪烁状态
                - 退出设置模式
                    - 触发条件
                        - 同时满足a和b
                            - a
                                - 安放高度已设置
                            - b
                                - APP蓝牙连接通知设备退出设置
                    - 指示灯
                        - 设备的指示灯进入蓝色灯常亮状态
            - 逗宠区域设置
                - 设置目的
                    - 逗宠期间镭射光会在设置的区域内移动，避免光点移动到宠物无法触及或造成危险/损失的区域
                - 进入设置模式
                    - 触发条件：a或b被满足
                        - a
                            - APP通过蓝牙连接通知设备进入，并得到设备确认
                        - b
                            - 逗宠区域未设置
                    - 初始光点位置
                        - 镭射光点位于0°点/中心点
                    - 发射状态
                        - 镭射光处于持续发射状态
                    - 指示灯
                        - 设备指示灯进入绿色闪烁状态，此时镭射光处在发射状态
                - 退出设置模式
                    - 触发条件
                        - 同时满足a、b和c
                            - a
                                - 在逗宠区域的4个顶点均已设置的前提下，
                            - b
                                - 四个顶点之间的距离均大于1米
                            - c
                                - APP蓝牙连接通知设备退出设置
                    - 指示灯
                        - 设备的指示灯进入蓝色灯常亮状态
                - APP通知设备将要调整位置的点
                    - 00
                        - 左上
                    - 01
                        - 右上
                    - 10
                        - 左下
                    - 11
                        - 右下
                    - 设备需自动将镭射光点移动到此顶点保存的位置，如果不存在保存的位置，则移动到0°点/中心点
                - 调整顶点位置
                    - APP通过蓝牙连接通知设备进行镭射光点位置调整，每次在一个方向上调整1度
                - 确认顶点位置
                    - APP通知设备结束当前顶点的位置设置
                    - 当4个顶点的位置均已设置时，需告知APP长度≤1米的边
            - 自动逗宠逻辑
                - 雷达监测到移动目标立刻启动逗宠逻辑
                - 连续30秒未监测到移动目标则退出逗宠逻辑，进入待机
                - 逗宠逻辑
                    - 计算频率：每0.05秒计算速度最大目标及其当前位置、运动方向、光点移动目标位置及速度
                    - 镭射光点位置
                        - 前导偏移+随机扰动法：目标速度＞0米/秒时：c = b + a x v + β x r
                        - 环绕试探+随机扰动法：目标静止时：C = a + R(θ) x d
                        - 边界处理：当光点触及逗宠区域边界时，按镜面反射原理调整运动方向
                        - 自主逃逸法：无论目标速度，周期性自主规划并执行逃逸：3 ：6 ：9 步
            - 开关机
                - APP通过蓝牙控制设备开机和关机
                - 关机时不关闭蓝牙模块
            - 上报逗宠记录
        - 数据处理
            - 缓存
                - 逗宠记录
                    - 数据结构
                        - 启动自动逗宠的时间戳，4字节
                        - 退出自动逗宠的时间戳，4字节
                - 逗宠区域
                    - 4个点的水平和垂直角度
                - 缓存不足时
                    - 优先删除最近的逗宠记录
                - 缓存上报
                    - 通过蓝牙成功上报后，自动删除对应的逗宠记录
            - 接口定义
                - 配对
                    - 设备上报设备SN给APP
                    - APP上报服务器：用户ID和设备SN
                - 建立连接时
                    - 手机告知设备时间
                - APP获取设备状态
                    - 开关机
                    - 逗宠区域是否设置
                    - 是否逗宠中
                        - 是，需要上报开始逗宠时间
                - APP请求
                    - 请求进入设置模式
                    - 请求结束设置模式
                    - 请求开始设置某顶点位置
                    - 请求调整顶点位置
                    - 请求结束顶点位置设置
                - 设备上报数据
                    - 逗宠记录
                        - 手机收到后上报服务器
                    - 上报长度小于1米的边或对角线
                        - 触发时机：APP请求结束顶点位置设置时，计算已存在边和对角线的长度
        - 物理UI
            - 指示灯
                - 熄灭
                    - 代表关机状态
                - 蓝色灯闪烁
                    - 代表没有蓝牙连接的待机状态
                - 蓝色灯常亮
                    - 代表有蓝牙连接的待机状态
                - 绿色灯闪烁
                    - 代表处在设置模式
            - USB供电口
            - 背部壁挂挂槽+蹲坐支架
